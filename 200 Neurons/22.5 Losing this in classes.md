[[12.5 Javascript|JavaScript]]

When an object is calling a method inside another method or function, `this` context is the "outside" function, therefore, `this` is not pointing to the object anymore.

```js
class Button {
  constructor(value) {
    this.value = value;
  }

  click() {
    alert(this.value);
  }
}

let button = new Button("hello");

setTimeout(button.click, 1000); // undefined
```

To fix that I have to make the method a class property of the arrow function type.

```js
class Button {
  constructor(value) {
    this.value = value;
  }
  click = () => {
    alert(this.value);
  }
}

let button = new Button("hello");

setTimeout(button.click, 1000); // hello
```
